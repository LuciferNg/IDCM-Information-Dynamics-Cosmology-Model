## 動機

標準 ΛCDM モデルは少なくとも6つの自由パラメータを必要とし、観測的緊張に直面している：$H_0$ 張力（5.0σ）、$S_8$ 張力（2.5σ）、DESI の動的暗エネルギー選好（2.5–3.5σ）。

IDCM は、宇宙は粒子の集まりではなく、固定点に収束する情報再帰であると提唱する。
