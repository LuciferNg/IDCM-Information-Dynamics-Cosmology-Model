## 動機

標準 ΛCDM モデルは少なくとも6つの自由パラメータ（$\Omega_m, H_0, \sigma_8, n_s, \Omega_b, \tau$）を必要とし、$H_0$ 張力（5.0σ）、$S_8$ 張力（2.5σ）、DESI の動的暗エネルギー選好（2.5–3.5σ）という観測的緊張に直面している。

IDCM は、**宇宙は粒子の集まりではなく、固定点に収束する情報再帰である**と提唱する。すべての観測的乖離はこの再帰の同調位相効果として自然に現れ、すべての宇宙定数は単一の二次方程式の代数的結果である。

## 核心メカニズム

### 生成方程式

$$x^2 + x - 1 = 0$$

この二次方程式は近似ではなく、モデルの生成核である。その正根：

$$\varphi^{-1} = \frac{\sqrt{5} - 1}{2} \approx 0.618033988749895$$

は黄金比の逆数であり、再帰の**固定点**である。

### 再帰過程

$$C_{n+1} = \frac{1}{1 + C_n}, \quad C_0 = 1$$

固定点におけるヤコビアン：

$$\lambda = \left|\frac{dC_{n+1}}{dC_n}\right|_{C=\varphi^{-1}} = \varphi^{-2} \approx 0.3819660113$$

$|\lambda| < 1$ により収束が保証される。8ステップ後の誤差は $10^{-3}$ 未満。
