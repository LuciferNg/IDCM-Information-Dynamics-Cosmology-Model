## 動機

標準 ΛCDM 模型最少要六個自由參數（$\Omega_m, H_0, \sigma_8, n_s, \Omega_b, \tau$），而且面對愈來愈嚴重嘅觀測張力：$H_0$ 張力（SH0ES 造父變星 vs Planck CMB，5.0σ）、$S_8$ 張力（Planck vs 弱透鏡巡天，2.5σ），同 DESI 對動態暗能量嘅偏好（$w_0$-$w_a$ 在 2.5–3.5σ）。呢啲張力暗示可能係系統誤差，或者模型本身有根本問題。

IDCM（資訊動力宇宙學模型）主張呢啲張力唔係量度錯誤——**宇宙唔係粒子嘅集合，而係一條收斂到固定點嘅資訊遞迴。** 所有觀測矛盾都係呢個遞迴嘅同步相位效應自然湧現出嚟，每個宇宙常數都係一條二次方程嘅代數結果。
