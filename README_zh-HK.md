## 核心機制

### 生成方程

$$x^2 + x - 1 = 0$$

正根 $\varphi^{-1} \approx 0.618034$ 係黃金比例倒數，亦係遞迴嘅固定點。

### 遞迴過程

$$C_{n+1} = \frac{1}{1 + C_n}, \quad C_0 = 1$$

收斂因子 $\lambda = \varphi^{-2} \approx 0.382 < 1$。八步後誤差低過 $10^{-3}$。
