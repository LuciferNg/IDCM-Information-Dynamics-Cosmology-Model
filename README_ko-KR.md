# IDCM — 정보동력우주모형 (Information Dynamics Cosmology Model)

[← 언어 선택으로 돌아가기](README.md)

---

**자유 매개변수가 0인 우주 모형.** 모든 상수는 단일 이차 방정식 $x^2 + x - 1 = 0$에서 재귀 $C_{n+1} = 1/(1 + C_n)$를 통해 생성됩니다. $H_0$ 장력(5.0σ → 동기 위상 효과)과 $S_8$ 장력(2.5σ → 해결됨)을 해결합니다. Δχ² = −9.8 (1853 데이터 포인트).

---

## 동기

표준 ΛCDM 모델은 최소 6개의 자유 매개변수가 필요하며, 관측적 긴장에 직면해 있습니다: $H_0$ 장력(5.0σ), $S_8$ 장력(2.5σ), DESI의 동적 암흑 에너지 선호(2.5–3.5σ).

IDCM은 우주가 입자의 집합이 아니라 고정점으로 수렴하는 정보 재귀라고 제안합니다.

| 기호 | 공식 | 값 | 기원 |

## 장력 해결

- $H_0$ (5.0σ): 🟡 동기 위상 효과로 설명
- $S_8$ (2.5σ): ✅ 해결됨, IDCM 예측 $S_8 = 0.786 \pm 0.008$
- DESI $w_0$-$w_a$ (2.5–3.5σ): ✅ 자연스러운 예측
