# IDCM — 정보 동역학 우주 모형 (Information Dynamics Cosmology Model)

**자유 매개변수가 없는 제1원리 우주 모형.** 모든 상수는 단일 2차 방정식 $x^2 + x - 1 = 0$에서 재귀 $C_{n+1} = 1/(1 + C_n)$를 통해 생성됩니다. **✅ 19개 표준 모형 매개변수 모두를 제1원리에서 예측합니다.** $H_0$ 긴장(5.0σ → 동기 위상 효과)과 $S_8$ 긴장(2.5σ → 해결됨)을 해결. Δχ² = −9.8 (1853 독립 데이터 포인트).

## 동기

표준 ΛCDM 모형은 최소 6개의 자유 매개변수($\Omega_m, H_0, \sigma_8, n_s, \Omega_b, \tau$)가 필요하며, $H_0$ 긴장(5.0σ), $S_8$ 긴장(2.5σ), DESI의 동적 암흑에너지 선호(2.5–3.5σ)라는 관측적 긴장에 직면해 있습니다.

IDCM은 **우주는 입자의 집합이 아니라 고정점으로 수렴하는 정보 재귀**라고 제안합니다.

## 핵심 구조

### 생성 방정식

$$x^2 + x - 1 = 0$$

양의 근 $\varphi^{-1} \approx 0.618034$는 황금비 역수이며 재귀의 **고정점**입니다.

### 재귀 과정

$$C_{n+1} = \frac{1}{1 + C_n}, \quad C_0 = 1$$

수렴 인자 $\lambda = \varphi^{-2} \approx 0.382 < 1$로 선형 수렴 보장.
